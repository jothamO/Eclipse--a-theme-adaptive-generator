# Eclipse Theme Adaptive Generator

Turn a single PNG into a theme-aware image that blends naturally across light, dim, and dark backgrounds.

## What it does

- Accepts an uploaded image and lets you crop it freely or to preset ratios (header or profile).
- Transforms the image into a theme-adaptive PNG using opacity and contrast mapping.
- Previews the result in real time against light, dim, and dark backgrounds.
- Exports one PNG you can use anywhere.

## How to use

1. Open `eclipse/index.html` in a modern browser.
2. Upload your source image.
3. Crop and choose a destination ratio (free, header, or profile).
4. Pick a style preset or adjust sliders manually.
5. Click **Download PNG** to save the adaptive result.

## Current status

Functional prototype. All core processing, previews, and export work in-browser without a backend.
